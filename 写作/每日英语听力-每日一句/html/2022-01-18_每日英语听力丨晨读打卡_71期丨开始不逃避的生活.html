晨读每日一句



Instead of wondering when your next vacation is, maybe you should set up a life you don't need to escape from.


与其期盼着你的下一个假期什么时候来临，不如开始一种你不需逃避的生活。




背景小知识

这句话出自美国著名营销大师、作家、演讲家赛斯·高汀，是一系列商业领域专著的作者，最有名的还是他的市场营销理念。他一共著作了《许可行销》（Permission Marketing）、《紫牛》（Purple Cow）等19本经典营销主题书籍，撰写了超过8000篇博客文章，并创办了最早期的在线营销公司YoyoDyne。高汀以其独特、鲜明和生动的演讲风格著称，并拥有一大批追随者。他的博客很受欢迎。


词汇点津

1. instead of 

代替；而不是…

Instead of 是个短语介词。在语言的实际运用中，instead of后几乎可连接各种词类及短语，其功能与连词十分相似。

例句：

We just had soup instead of a full meal.

我们没有吃全餐，只喝了汤。

Now I can walk to work instead of going by car.

现在我可以步行去上班，而不必开车了。

The islanders use a system of barter instead of money. 

岛上的居民实行的是以物易物，而不是用货币。


2. set up

v. 建立；装配；开业；竖立；引起，引发；产生

例句：

The two sides agreed to set up a commission to investigate claims.

双方同意成立一个委员会，调查索赔问题。

The government announced the setting up of a special fund.

政府宣布设置了一项特别基金。

I set up the computer so that they could work from home. 

我把电脑设置好，这样他们就可以在家办公了。


近义词&词组辨析：

set up 侧重指开始着手创建公司、机构、委员会等。

build 普通用词，应用范围最广，既可用于表示建造具体有形的建筑，又可用于表示抽象意义上的建立、构建。

found 常指通过提供必要的资金创建城市、公司、学校等，含打下基础之意。

establish 强调稳固地建成，可指建立国家、政府、学校或商店等，也可指建立信仰、名誉、法律、制度、关系等。

construct 强调按照一定设计建造规模较大、结构较复杂的物体，特别指按照图纸构筑某种建筑物。
