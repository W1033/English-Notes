晨读每日一句



No one saves us but ourselves, no one can and no one may. We ourselves must walk the path.

除了自己，没人可以、也没人会拯救我们。我们的路必须由我们自己来走。


背景小知识

这句话出自释迦牟尼，佛教创始人。姓乔答摩，名悉达多。“释迦牟尼”（Śākyamuni）是佛教徒对他的尊称。释迦是种族名，意为“能”；牟尼，亦译为“文”，是尊称，意为“仁”、“儒”、“忍”、“寂”。合为“能仁”、“能儒”、“能忍”、“能寂”等，意即释迦族的“圣人”。


词汇点津

1. save / seɪv /

v.救助，搭救；避免，减少；积攒，储蓄；节省（钱、时间或其他资源）；保存，保留；收集；（为某人）保留；（计算机）保存（数据）；（足球等）救球，防止对方进球

n.救球；保持（其他投手取得的领先地位）；（计算机）存储数据

prep.<正式> 除……之外

例句：

The goalie brought off a superb save.

守门员作出了一次精彩的扑救。

A new filter can save on energy bills.

一个新的过滤器能节省能源开支。

There is no incentive for people to save fuel.

没有使人们节约燃料的鼓励办法。

They are fighting to save what is left of their business.

他们在拼命抢救他们仅余的业务。

常见用法：

save energy 节约能源

save electricity 节约用电

save food 节约粮食

save money 省钱

save on 节省，节约；节省

save for 储蓄；为…存着

save up 储蓄；贮存

save from 保留，免于；使免遭

save one's life 挽救某人生命

save one's energy 节省某人的力气

口语表达：

save your breath 别唠叨了；别白费口舌了

同根词：

saving adj.节约的；挽救的；补偿的；保留的

saved adj.获救的；已保存的

saving n.节约；挽救；存款

saver n.救助者；节俭的人；节约装置

saving prep.考虑到；除...之外


近义词辨析：

▲save：指为了某种理由或后续的使用所进行的积攒、存储。

例：I save on fares by walking to work.

我步行上班，可以省车钱。


▲store : 指储存、储藏。多用于现实存在的东西的存储，以备将来使用。

例：The server is designed to store huge amounts of data.

该服务器是为存储大量数据设计的。


▲keep : 指存放、保留。强调的是不丢弃、不放弃。

例：We keep a tape of incoming calls.

我们保留打进电话的录音磁带。

2. path 🇬🇧/ pɑːθ / 🇺🇸/ pæθ /

n.小径，小道；（开出的）通道；（事物或人移动的）路线，轨迹；（有助于实现某事的）道路，途径；通勤火车时刻表；（计算机）路径

例句：

A group of reporters blocked his path.

一群记者堵住了他的路。

At this point the path descends steeply.

小路从这里陡然而下。

The path intersected with a busy road.

小路与一条繁忙的大路相交。

常见用法：

career path 职业道路

flight path 飞行路线；航线

beaten path 常规；惯例；常走的路

propagation path 传播路径，传播通路

winding path 曲径；绕组分支

同根词：

pathless adj.无路的；人迹罕到的
