晨读每日一句


You will never have what you like until you learn to like what you have.

要得到你喜欢的东西，应先学会喜欢你已有的东西。


背景小知识

这句话出自约翰·沃尔夫冈·冯·歌德（Johann Wolfgang von Goethe），他出生于美因河畔法兰克福，德国著名思想家、作家、科学家，他是魏玛的古典主义最著名的代表。而作为诗歌、戏剧和散文作品的创作者，他是最伟大的德国作家之一，也是世界文学领域的一个出类拔萃的光辉人物。他在1773年写了一部戏剧《葛兹·冯·伯利欣根》，从此蜚声德国文坛。





词汇点津

1. never / ˈnevər /

adv.决不，永不；不会，不可能（表示惊讶或震惊）


例句：

Her determination never wavered.

她的决心从未动摇过。

She promised to never reveal the secret to anyone. 

她承诺永远不会向任何人泄露这个秘密。

Discussions never progressed very far.

讨论从未有太大进展。


常见用法：

never forget 永不忘记

never do 绝对不行，绝不可以

never mind 没有关系，不要记在心上

never before 以前从未曾有过

never ever 绝不，从不

almost never 几乎从不

better late than never 迟做比不做好

never so 非常；空前地

never fail to 一定会...

now or never 勿失良机，机不可失；把握现在

as never before 前所未有地，空前地

tomorrow never comes 切莫依赖明天

never for a moment 决不

never heard of it 从来没听说过

never so much as 甚至连…都不


同根词：

nevermore adv.决不再；永远不再


★ 小拓展

表示“从不，任何情况下都不”的表达：

not at any time 从未

at no time 决不，从不

on no occasion 决不

under no circumstances 在任何情况下都不

2. learn / lɜːrn /

v.学习，学会；得知，获悉；背熟，熟记；认识到，（从……）吸取教训

例句：

I must learn to bite my lip.

我必须学会不随便说话。

When do babies learn to discriminate voices?

婴儿什么时候学会辨别嗓音呢？

They learn new roles and unlearn old ones.

他们学习新的角色，试图忘记旧的。


常见用法：

learn from 向……学习

learn about 了解；学习

learn by oneself 自学

learn of 听说，听到；获悉

learn to do 做事；学会做某事；学习做……

learn by heart 记住；默记；背诵

learn a lesson 受到教训

motivation to learn 学习的动机

learn the ropes 摸到门道；知道事情的内情

learn the hard way 通过艰难困苦而学到

learn and live 活到老学到老；学无止境


词根：learn

同根词：

learned adj.博学的；有学问的；学术上的

learnedly adv.学识上

learning n.学习；学问

learner n.初学者，学习者


近义词辨析：

learn：强调知识或技能的获取，可以通过不同的方式，包括学习、观察、体验等。

例：Children learn best when they are actively engaged in hands-on activities.

孩子们在积极参与动手活动时学得最好。


study：强调有目的地、系统地学习，通常涉及阅读、研究、练习等有组织的努力。

例：These proposals deserve careful study.

这些建议值得认真研究。
