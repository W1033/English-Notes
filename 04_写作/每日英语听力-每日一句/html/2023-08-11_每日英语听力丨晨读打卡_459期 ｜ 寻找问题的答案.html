晨读每日一句


We learn more by looking for the answer to a question and not finding it than we do from learning the answer itself.

为了寻找一个问题的答案而没找到它的过程，要比得到答案本身所学到的还要多。


背景小知识

这句话出自罗伊·亚历山大（Lloyd Alexander），是美国儿童文学、奇幻文学作家，代表作是《不列颠编年史》。1985年参与编剧电影《黑神魔》。在1971、1982年获得美国国家书卷奖。





词汇点津

1. learn / lɜːrn /

v.学习，学会；得知，获悉；背熟，熟记；认识到，（从……）吸取教训；<旧>教导（某人）

例句：

Children do not learn at an even pace.
孩子学东西时快时慢。
I must learn to bite my lip.
我必须学会不随便说话。
He needs to learn to curb his temper.
他得学着控制自己的脾气。


常见用法：

learn from 向…学习

learn about 了解；学习

learn by heart 记住 ; 背诵 ; 背熟 ; 牢记

learn by oneself 自学

learn of 听说，听到；获悉

learn by heart 记住；默记；背诵

learn a lesson 受到教训

learn the ropes 摸到门道；弄清内幕

learn the hard way 通过艰难困苦而学到

learn and live 活到老学到老；学无止境


同根词：

learned adj.博学的；有学问的；学术上的

learnedly adv.学识上

learning n.学习；学问

learner n.初学者，学习者


近义词辨析：

learn : 常指一般性的学习，尤指初级阶段的学习，表示摹仿实践的过程；着重成果时，作“学会”解。

例：How did you learn to bake cakes?

你是怎么学会烤蛋糕的？


study : 着重指学习过程和高深的或周密的研究，尤指从书本上学习。

例：This study builds on earlier work.

这项研究是在以往工作的基础上进行的。

2. answer / ˈænsər /

v.回答，答复；回复；接电话；应门；答（题）；回应，反击；满足，适合；与……相符，相配；解决（问题）

n.回答，答复；答案；解决办法；回信，回函；接听，应门；相当的人，对应物

例句：

（作名词）Her answer amounted to a complete refusal.

她的答复等于完全拒绝。

（作动词）He couldn't answer the question.

他没有能回答那个问题。


常见用法：

in answer to 回答；应…要求

answer for 因……而受罚；对……负责

immediate answer 立即应答

answer back 应答，回复；顶嘴

know all the answers 见多识广；自命不凡

soft answer 温和的回答；委婉的回答

answer the needs of 满足...需要

answer up 迅速回答
