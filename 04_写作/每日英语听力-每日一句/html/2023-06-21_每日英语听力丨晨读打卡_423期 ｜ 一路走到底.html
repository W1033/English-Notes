晨读每日一句

The best way out is always through.

找到出口的最好方式就是一路走到底。


背景小知识

今天的这句话旨在鼓励大家不轻言放弃，不畏前行。出自罗伯特·弗罗斯特（Robert Frost），出生于美国旧金山，20世纪最受欢迎的美国诗人之一。代表作品有《诗歌选集》、《一棵作证的树》、《山间》、《新罕布什尔》、《西去的溪流》和诗剧《理智的假面具》、《慈悲的假面具》、《诗歌全集》等。


词汇点津

1. always / ˈɔːlweɪz /

adv.总是，每次都是；一直，长久以来；永远；老是，一再；总还有（表示有另一种可能）

例句：

I'm always running myself down.

我总是深深自责。

I've always worked in education.

我一直从事教育工作。

She's always cracking jokes.

她总爱说笑话。


常见用法：

as always 同平时一样

always the case 常常如此

since always 一向，一向以来，一直

\ | /

★

2. through / θruː / 

prep.穿过，从一端到另一端；透过（玻璃、窗户等）；越过（地区或空间），在……之间；以，凭借；贯穿，遍及；自始至终，从头至尾；彻底地；经历，度过；胜利完成；（提议、观点）获得通过，得到批准

adv.从一端到另一端；完全地，彻底地，十足地；接通电话；通过；直达，迳直；从头至尾；到（某个阶段或某种程度）；胜利完成

adj.完成的，结束的；直达的，直通的，中途不停的；断绝的；（公路）对穿的，双向的

例句：

Most advertisements work through suggestion.

多数广告都是通过暗示而发挥作用。

We'll muddle through somehow.

我们能想办法应付过去。

He's British through and through.

他是地地道道的英国人。

The deal did not go through.

这笔交易未谈成。


常见用法：

go through 参加；经受；仔细检查；被通过

go through with 完成，实行；把…进行到底

all through 一直；始终

through with 完成

through and through 完全；彻头彻尾地

live through 经受住 ; 经历过

see through 看透 ; 看穿 ; 识破 

get through 接通电话 ; 结束 ; 到达

fall through 落空 ; 失败 ; 毁灭 ; 告吹

look through 浏览 ; 详尽核查 ; 仔细查看

break through 突破 ; 突围 ; 冲破

come through 经历 ; 获得成功 ; 成功
